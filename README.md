SCPD Video Downloads
--------------------

Again, usage by

    scpd-curl

Dependencies

- Node v0.8+
- MPlayer ~v1.1
- Install this package using

        npm install -g scpd-curl

- If you are a developer, install other node deps using

        npm install

  in the `scpd/` directory
